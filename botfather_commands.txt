start - Начать работу с ботом и получить приветственное сообщение
auth - Авторизоваться в Facebook для доступа к рекламным аккаунтам
accounts - Показать список доступных рекламных аккаунтов
campaigns - Показать список кампаний для выбранного аккаунта
ads - Показать список объявлений для выбранной кампании
menu - Открыть главное меню бота
help - Показать список всех доступных команд и инструкции
